调色图层批量合并 0.3.0 测试说明

插件用途

这是一个 Photoshop 调色图层批量合并工具。
当前版本可先生成 Dry Run，也可创建完整文档副本后执行自动解锁、隐藏剪贴层清理与逐层向下合并。
原始文档不会被修改。
面板会自动保存最新诊断日志，并提供“导出日志”按钮。
0.2.2 修复确认按钮未继续执行、日志文件选择器权限缺失，以及普通图层被误执行解锁的问题。
0.2.3 彻底移除弹窗与等待式审批，改为面板内固定确认卡；Dry Run 不再要求锁定审批。
0.3.0 支持递归展开组中组，使用结构定位器解析副本目标，并在完成后自动隐藏原始调色源层。

兼容范围

· Photoshop 25.0.0 或更新版本
· 插件运行时不需要联网
· 本构建已完成逻辑测试、Manifest 校验和 CCX 结构校验
· 尚需使用真实 PSD 验证不同 Photoshop 版本的剪贴蒙版、背景层和锁定属性兼容性

推荐安装

1. 退出 Photoshop。
2. 双击 Dino-color-layer-batch-merge_PS.ccx。
3. 在 Adobe 插件安装窗口中确认安装。
4. 重新打开 Photoshop。
5. 从“增效工具 / Plugins”菜单打开 调色图层批量合并。

使用方法

1. 在图层面板中选择一个或多个调色层，点击“记忆当前所选”。
2. 选择多个目标层，或选择一个目标图层组，再点击第二个“记忆当前所选”。
3. 点击“生成 Dry Run”，核对日志里的处理顺序。
4. 点击“在安全副本中实际合并”。
5. 如果检测到锁定层，审批是否允许在文档副本中自动解锁。
6. 确认生成的新副本文档及执行日志。

测试重点

· 多选调色层能否按图层面板顺序记录
· 目标组能否展开直接子图层
· 组内剪贴蒙版是否不会被误当成独立目标
· 目标正上方连续剪贴链能否被正确识别
· 隐藏剪贴层是否被计划为丢弃
· 可见剪贴层是否按从下往上的顺序计划向下合并
· 调色层是否保持原堆叠顺序复制，并从下往上逐层向下合并
· 锁定层是否只弹出一次审批窗口
· 原始文档在实际执行前后是否完全没有变化
· 新副本中的隐藏剪贴层是否被删除
· 组中组能否递归展开并正确定位到副本文档中的目标层
· 完成后副本文档中的原始调色源层是否自动隐藏
· 背景层或锁定层经审批后能否自动解锁并继续
· 审批确认层的两个按钮能否立即关闭确认层并继续流程
· “导出日志”能否生成包含调用开始、完成或失败检查点的 .log 文件

已知限制

· 不支持把嵌套组本身当成最终合并目标；目标组会递归展开其中的可处理图层。
· 不同 Photoshop 版本的背景层转换与锁定字段可能存在兼容差异。
· 执行失败时插件会尝试回滚副本中的本次历史步骤；原始文档不受影响。

反馈时请提供

· macOS 或 Windows 版本
· Photoshop 完整版本号
· 安装是否成功
· 面板和 Dry Run 日志截图
· 实际图层结构截图
