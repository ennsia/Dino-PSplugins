# Layer Quick Jump 0.2.0

First Dino-branded release of the expanded layer anchor workflow.

## Included

- `Dino-layer-quick-jump_PS.ccx`: recommended installer
- `README_TEST_CN.txt`: Chinese installation, usage, and testing notes
- `layer-quick-jump-v0.2.0-offline.zip`: complete offline package
- `SHA256SUMS.txt`: artifact checksums

## Changes

- Shows only the saved layer or group name for faster reading.
- Adds a small clear button for each record.
- Adds Transfer to move the current multi-layer selection below an anchor.
- Preserves the selected layers' document order during transfer.
- Prevents moving an anchor or a group containing the anchor onto itself.
- Uses a more compact three-action layout.

## Verified

- Installed and tested locally with Photoshop 2025 version 26.11.0.
- Logic and UXP panel simulation tests pass.
- CCX package structure and SHA256 checks pass.
