# 调色图层批量合并

Photoshop UXP 插件，用于把一组调色剪贴层逐层应用到多个目标图层。

## 当前版本：0.3.0 安全副本执行版

- 记忆单个或多个调色层。
- 记忆多个目标层，或展开所选图层组的直接子图层。
- 分析目标正上方连续的剪贴蒙版链。
- 计划丢弃隐藏剪贴层，并按从下往上的顺序逐层向下合并可见剪贴层。
- 检查目标与剪贴层的锁定状态，以单次弹窗请求自动解锁审批。
- 输出完整日志。
- 自动将最新诊断日志写入插件数据目录，并可从面板导出 `.log` 文件。
- 支持 Dry Run。
- 实际执行前自动创建完整文档副本，只在副本中解锁、删除和逐层合并。
- 图层组选作目标时递归展开任意层级的嵌套组。
- 合并完成后自动隐藏副本文档中的原始调色源层，避免画面重复叠加产生色差。
- 原始文档不会修改。

## 加载

1. 打开 Adobe UXP Developer Tool。
2. Add Plugin，选择本目录。
3. Load 后，从 Photoshop 的“增效工具 / Plugins”菜单打开“调色图层批量合并”。

## 实际执行

1. 记录调色层与目标层。
2. 建议先点击“生成 Dry Run”核对顺序。
3. 点击“在安全副本中实际合并”。
4. 审批自动解锁后，插件创建副本并执行。

## 当前限制与测试重点

- 不同 Photoshop 版本对背景层转换、剪贴属性和锁定属性的返回值可能存在差异，首次测试请使用可舍弃的 PSD。
- 执行失败时会尝试回滚副本中的本次历史步骤，并在日志中保留失败位置。
- 0.2.2 使用固定确认按钮与单一等待器，避免 UXP 动态事件未唤醒审批 Promise。
- 0.2.2 补充文件选择器权限，修复诊断日志无法导出。
- 仅对规划中确实锁定的层执行解锁，避免普通图层收到不必要的锁定命令。
- 0.2.3 彻底删除 Promise 式审批和弹窗；实际执行改为普通面板确认卡，点击批准后直接进入执行引擎。
- Dry Run 只报告锁定情况，不再要求无意义的解锁审批。
- 0.3.0 增加嵌套组递归展开、结构定位器和调色源自动隐藏。
- 面板最小宽度降至 260 像素，改善窄侧栏停靠适配。侧栏折叠/固定展开仍由 Photoshop 顶部的 `<<` 控制。
