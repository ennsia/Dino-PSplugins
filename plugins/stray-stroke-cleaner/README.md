# Stray Stroke Cleaner

Prototype Photoshop UXP panel for reviewing cleanup workflows around accidental or stray strokes.

## Current Features

- Shows the active document name.
- Counts top-level layers in the active document.
- Creates a non-destructive review layer for cleanup notes.

## Load in Photoshop

1. Open Adobe UXP Developer Tool.
2. Add this folder as a plugin.
3. Load it into Photoshop.
4. Open the panel from Photoshop's Plugins menu.

## Roadmap

- Add image analysis helpers.
- Add configurable detection thresholds.
- Preview suspected stray strokes before changing the document.
