# Layer Quick Jump CEP 安装说明

## 本地离线安装

1. 解压离线包。
2. 退出 Photoshop。
3. 双击 `INSTALL_CEP_LOCAL.command`。
4. 如果 macOS 阻止运行，请右键点击脚本，选择打开。
5. 重新打开 Photoshop。
6. 在 `Window > Extensions` 或扩展面板菜单里查找 `Layer Quick Jump CEP`。

## 卸载

双击 `UNINSTALL_CEP_LOCAL.command`。

## 说明

- 这个版本是 CEP 离线安装实验版。
- 安装脚本会开启 CEP debug mode，用于加载未签名的本地扩展。
- 如果 CEP 版能显示，后续离线包优先走 CEP 路线。
